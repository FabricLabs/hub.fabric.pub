# Dependency trees

Generated by **`npm run report:dependency-trees`** (`scripts/generate-dependency-trees.js`).

- **`* .json`** — full tree from `npm ls --json --all` (programmatic).
- **`* .txt`** — `npm ls --all` ASCII output from npm.
- **`index.json`** — which repos were scanned and paths on disk.

Override paths: **`FABRIC_CORE`**, **`FABRIC_HTTP`**, or **`DEP_TREE_PROJECTS`** (comma-separated roots).
